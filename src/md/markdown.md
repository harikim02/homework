# 마크다운(Markdown) 문법 정리

Markdown은 글을 꾸미는 간단한 방법임
특수한 프로그램 없이도 글을 예쁘게 정리할 수 있음

## 1. 제목 만들기

'#' 개수로 글자 크기를 정함. 6단계까지 가능함.
'#' 개수가 많을 수록 글씨 크기가 작아짐.

## 2. 글자 꾸미기

글자크기를 굵게 하는 건 글자 앞 뒤로 \**을 넣어줌.
글자를 기울게 하는 건 글자 앞 뒤로 *을 넣어줌.
글자에 취소선을 만들게 하는 건 글자 앞 뒤로 ~~을 넣어줌.

## 3. 목록 만들기

### 순서 있는 목록

1. 첫 번째
2. 두 번째
3. 세 번째

### 순서 없는 목록

- 사과
- 바나나

* 오렌지

## 4. 코드

### 한 줄 코드 (Inline Code)

글 중간에 짧은 단어나 명령어를 넣을 때 사용 : 백틱( ` ) 1개

### 여러 줄 코드 블록 (Code Blocks)

여러 줄의 소스 코드를 한꺼번에 보여줄 때 사용 : 백틱 3개( ``` )로 코드의 시작과 끝에 사용
백틱 3개 바로 옆에 java, python, html 등 언어 이름을 적으면, 코드의 문법에 맞춰 색상이 입혀짐.

## 5. 인용문

> 인용문
>
> > 중첩 인용문 가능

> 주의: ``` 뒤에 언어 이름을 쓰면 코드 색이 더 예쁘게 나와요.

## 5. 링크

[] → 설명
() → 링크 주소
[구글 바로가기](https://www.google.com)

## 6. 이미지

! → 이미지 표시
[] → 설명
() → 이미지 주소
![강아지](https://example.com/dog.jpg)
